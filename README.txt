
Aylan Beauty - Demo Template (YouCan-ready)
===========================================

ملف القالب التجريبي متاح للتعديل والرفع داخل YouCan.
الملفات الموجودة:
 - index.html     -> الصفحة الرئيسية
 - product.html   -> صفحة منتج مفصلة
 - about.html     -> صفحة عن البراند
 - contact.html   -> صفحة تواصل
 - style.css      -> أنماط التصميم
 - script.js      -> تفاعلات بسيطة (سلة تجريبية، slider)
 - assets/        -> شعار وملفات صور مؤقتة (SVG)

طريقة المعاينة محلياً:
1) حمّل جميع الملفات.
2) افتح index.html بمتصفحك (Chrome/Firefox).
أو استعمل Live Server داخل VSCode.

ملاحظات للرفع إلى YouCan:
- YouCan يسمح بتعديل القالب عبر المصمم البصري. يمكنك رفع الصور من مجلد assets واستخدام النصوص الموجودة في HTML كنصوص جاهزة.
- محتوى المنتجات مؤقت (Placeholder) — عندما تكون جاهزًا، استبدل النصوص والصور بالمنتجات الحقيقية عبر لوحة تحكم YouCan.

لو بغيت، نقدر نحول هذا القالب إلى ZIP جاهز للتحميل وندّلك رابط التنزيل.
